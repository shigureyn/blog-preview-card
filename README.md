## What I learned

During this project, I practiced building a small responsive card component using semantic HTML and clean CSS structure.

I also practiced:

* Using the `article` element for a standalone card
* Connecting local fonts with `@font-face`
* Working with spacing, border radius, and box-shadow
* Creating a responsive layout for mobile and desktop screens
* Organizing CSS with BEM-style class names
